# 🧮 Calculator Project

A simple calculator built using HTML, CSS, and JavaScript.

## ✨ Features
- Basic arithmetic operations (+, −, ×, ÷)
- Real-time calculation
- Clear (AC) button
- Keyboard support
- Responsive design

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
